Git

  • git init — создаёт Git-репозиторий в текущей папке (инициализация).
  • git clone <URL> — скачивает удалённый репозиторий к себе на компьютер.
  • git status — показывает состояние рабочей директории и staging area: какие файлы изменены, добавлены и т.д.
  • git add <file> — добавляет указанный файл(ы) в индекс (подготовка к коммиту).
  • git add . — добавляет все изменения в текущей директории в индекс.
  • git commit -m "Сообщение" — фиксирует изменения как новый коммит с описанием.
  • git log — показывает историю коммитов. Добавьте --oneline для компактного вида.
  • git branch — список веток. git branch new-branch — создать новую ветку.
  • git checkout <branch> — переключиться на указанную ветку.
  • git switch <branch> — современная альтернатива checkout (Git ≥2.23).
  • git merge <branch> — объединяет указанную ветку с текущей.
  • git push origin <branch> — отправляет локальные коммиты в удалённую ветку.
  • git pull — скачивает и объединяет изменения из удалённого репозитория (git fetch + git merge).
  • git remote add origin <URL> — подключает удалённый репозиторий под именем origin.
  • git stash — временно сохраняет незакоммиченные изменения, чтобы можно было переключиться на другую ветку.
